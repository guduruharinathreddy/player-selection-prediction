# Player Selection Prediction

## Description
This project focuses on predicting player selection using Machine Learning techniques.
The model analyzes player performance data to assist in selection decisions.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
- Data collection and analysis
- Data preprocessing
- Feature selection
- Model training
- Model evaluation

## How to Run
1. Download the repository
2. Open the .ipynb file in Jupyter Notebook
3. Run all cells
